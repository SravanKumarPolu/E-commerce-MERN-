Manual Authentication Test Steps

🧪 Test Scenarios

1. Test Unauthenticated User

  1. Open frontend in incognito mode
  2. Go to any product page
  3. Try to select a color and click "Add to Cart"
  4. ✅ Expected: Should show "Please login" message and redirect to login
  5. ✅ Expected: Should NOT add item to cart

2. Test Cart Access Without Login

  1. While not logged in, go to /cart
  2. ✅ Expected: Should redirect to login page
  3. ✅ Expected: Should show "Please Login" message

3. Test Authentication Flow

  1. Go to login page
  2. Create a new account or login
  3. ✅ Expected: Should redirect to home page
  4. ✅ Expected: Navbar should show profile dropdown

4. Test Add to Cart After Login

  1. After logging in, go to a product page
  2. Select a color and click "Add to Cart"
  3. ✅ Expected: Should add item to cart successfully
  4. ✅ Expected: Should show "Item added to cart" message
  5. ✅ Expected: Cart count should increase

5. Test Logout

  1. Click profile dropdown and logout
  2. ✅ Expected: Should clear cart and redirect to login required state
  3. Try to access cart page
  4. ✅ Expected: Should redirect to login
